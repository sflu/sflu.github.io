给生活来点不一样

刚刚搭建了博客，解决了如下几个问题：

  1. 使用hugo搭建本地博客
  2. 将博客上传到github
  3. 使用CNAME将lusf.site域名指向sflu.github.io

更换主题

刚开始用hugo官方推荐的ananke主题，感觉有点太简单了，后来用了飞雪无情推荐的Maupassant.

  • 添加主题命令
    • git submodule add https://github.com/rujews/maupassant-hugo themes/maupassant

坑

  • 添加其他公钥

    • 使用不同的github账号时，需要先为新账号生成新公钥；并在~/.ssh/config中为不同用户指定不同密钥文件。
    • 添加remote url时，需要git remote set-url origin git@github.com:sflu/blog.git 或者git remote add origin git@github.com:sflu/blog.git，注意git@github.com前缀。
