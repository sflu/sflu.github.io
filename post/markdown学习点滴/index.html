表格

– 表格基础

| 左对齐标题 | 右对齐标题 | 居中对齐标题 |
| :------| ------: | :------: |
| 短文本 | 中等文本 | 稍微长一点的文本 |
| 稍微长一点的文本 | 短文本 | 中等文本 |

对应效果：

左对齐标题 右对齐标题 居中对齐标题
短文本 中等文本 稍微长一点的文本
稍微长一点的文本 短文本 中等文本
  • 列宽

MarkDown表格最常用的技巧莫过于列宽控制了，如果不满意某列的列宽，只需要在表格前加上(有些Markdown解析器可能不支持) ：

<style>
table th:nth-of-type(1) {
    width: 100px;
}
</style>

其中1为要控制列宽的列序号，示例如下：
